# Dataset-TCs
1. 'Dataset-TCs' contains a total of 24 data matrices corresponding to the core experimental datasets used in this study.
2. 'ExAxis' and 'EmAxis' represent the excitation and emission wavelength ranges, respectively.
3. 'Blank_01', 'Blank_02', and 'Blank_03' correspond to blank background measurements.
4. 'C01' â€“ 'C07' represent the calibration set samples.
5. 'P00_1', 'P00_2', and 'P00_3' correspond to real (unspiked) samples.
6. 'P01', 'P02', and 'P03' correspond to spiked samples used as the prediction set.
7. 'TCex', 'TCem', 'OTCex', 'OTCem', 'CTCex', 'CTCem' represent the excitation and emission spectra of tetracycline (TC), oxytetracycline (OTC), and chlortetracycline (CTC) reference standards, respectively.
All datasets were collected under the same experimental conditions as described in the Materials and Methods section and were used for EEM data analysis and PARAFACM modeling in this work.
